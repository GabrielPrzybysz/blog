About

Hey, I'm Gabriel Przybysz (aka perxibes). I’m a security engineer who spends a lot of time around cloud infrastructure and production systems, mostly trying to understand why things break and how to make them slightly less surprising next time.

Why this blog exists

Security and infrastructure work generates a lot of "wait, that actually happened?" moments that are worth documenting. This blog is where I write about:

  • ▸ Vulnerabilities I've found and how I found them
  • ▸ Infrastructure configs that saved prod (and some that almost killed it)
  • ▸ Pentesting techniques and methodologies that actually work
  • ▸ Post-mortems from production incidents
  • ▸ Security tooling and automation I've built to make life easier
